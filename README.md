# SQL Practice – 50 Questions (Beginner to Advanced)

## 📌 Overview

This repository contains **50 SQL practice questions** solved using structured datasets.
The goal of this project is to strengthen SQL skills required for **Data Analyst roles**, covering concepts from **basic queries to advanced analytical SQL**.

The dataset includes employee information, order transactions, and department details to simulate real-world business scenarios.

## 🗂️ Datasets Used

### 1. Employees Table

Contains employee details such as:

* `emp_id`
* `name`
* `department`
* `city`
* `salary`
* `join_date`

### 2. Orders Table

Contains order transactions:

* `order_id`
* `emp_id`
* `amount`
* `order_date`

### 3. Departments Table

Contains department information:

* `dept_id`

### 🟢 Basic SQL

### 🟡 Intermediate SQL

* Subqueries
* Finding duplicates
* Ranking employees

### 🔵 JOIN Operations

* INNER JOIN
